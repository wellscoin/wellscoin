What is Wellscoin?
----------------

Wellscoin is based on Litecoin/Bitcoin using scrypt as a proof-of-work algorithm.
 - 5 minute block targets
 - subsidy halves in 160k blocks
 - ~16 million total coins
 - 16 coins per block
